## 1. 安装并运行

```bash
cp .env.example .env.local
npm install
npm run dev
```

## 2. 创建页面

为所有已配置语言创建页面：

```bash
npm run typematter -- new --all-languages --slug guides/install --title.en "Install" --title.cn "安装" --section.en "Guides" --section.cn "指南"
```

只创建单个语言：

```bash
npm run typematter -- new --lang cn --slug guides/install --title "安装" --section "指南"
```

## 3. 替换示例内容

你的 MDX 源文件位于 `content/`。

```text
content/
  cn/
    index.mdx
    guides/
      install.mdx
  en/
    index.mdx
    guides/
      install.mdx
```

## 4. 更新导航

`nav.config.ts` 控制分组、显示顺序、隐藏项和外链。它不创建页面，MDX 文件才创建页面。

```ts
{ type: "doc", slug: "guides/install" }
```

<Callout type="note" title="必填 frontmatter">
默认校验器要求每个页面包含 `title`、`order`、`section`、`type` 和 `description`。
</Callout>

## 5. 校验

```bash
npm run validate:docs
```
