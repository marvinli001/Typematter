Typematter 会把站点静态导出到 `out/`。部署这个目录即可。

## 1. 配置生产 URL

```bash
cp .env.example .env.local
```

设置：

```bash
TYPEMATTER_SITE_URL=https://docs.example.com
NEXT_PUBLIC_SITE_URL=https://docs.example.com
```

`TYPEMATTER_SITE_URL` 用于生成 `robots.txt` 和 `sitemap.xml`。

## 2. 校验与构建

:::steps
1. 运行 `npm run validate:docs`。
2. 运行 `npm run build`。
3. 用 `npm start` 预览。
4. 部署 `out/`。
:::

## 可选 Ask AI

不启用 Ask AI 时，文档站仍是完整静态站点。要启用 AI 页签，部署 `integrations/cloudflare-ask-ai-worker/` 中的 Worker，并设置：

```bash
NEXT_PUBLIC_TYPEMATTER_ASK_AI_ENDPOINT=https://your-worker-domain.workers.dev
NEXT_PUBLIC_TYPEMATTER_ASK_AI_TIMEOUT_MS=25000
```

<Details summary="生成文件">
  构建产物包含 `out/typematter/search/`、`out/typematter/ask-index.json`、`out/robots.txt` 和 `out/sitemap.xml`。
</Details>
