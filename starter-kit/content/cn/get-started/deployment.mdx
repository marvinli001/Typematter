Typematter 当前默认使用 Next.js 静态导出模式，构建产物位于 `out/`。

## 推荐流程

::: steps
1. 设置 `TYPEMATTER_SITE_URL=https://docs.example.com`。
2. 执行 `npm run build`。
3. 将 `out/` 目录部署到任意静态托管平台。
:::

## 适合的托管方式

<Cards columns="3" mobileColumns="1">
  <Card title="Vercel" description="直接构建并发布静态输出。" />
  <Card title="Cloudflare Pages" description="适合全球静态分发。" />
  <Card title="Nginx / OSS / S3" description="只要能托管 HTML/CSS/JS 即可。" />
</Cards>

<Details summary="为什么需要 TYPEMATTER_SITE_URL">
  它用于生成 `robots.txt` 和 `sitemap.xml` 里的完整绝对地址。
</Details>
