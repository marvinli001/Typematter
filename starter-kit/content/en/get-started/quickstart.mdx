## 1. Install and run

```bash
cp .env.example .env.local
npm install
npm run dev
```

## 2. Create a page

For every configured language:

```bash
npm run typematter -- new --all-languages --slug guides/install --title.en "Install" --title.cn "安装" --section.en "Guides" --section.cn "指南"
```

For one language:

```bash
npm run typematter -- new --lang en --slug guides/install --title "Install" --section "Guides"
```

## 3. Replace the sample content

Your MDX source lives under `content/`.

```text
content/
  en/
    index.mdx
    guides/
      install.mdx
  cn/
    index.mdx
    guides/
      install.mdx
```

## 4. Update navigation

`nav.config.ts` controls grouping, display order, hidden items, and external links. It does not create pages; MDX files do.

```ts
{ type: "doc", slug: "guides/install" }
```

<Callout type="note" title="Required frontmatter">
Every page needs `title`, `order`, `section`, `type`, and `description` for the default validator.
</Callout>

## 5. Validate

```bash
npm run validate:docs
```
