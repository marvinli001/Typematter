Typematter exports a static site into `out/`. Deploy that directory.

## 1. Configure production URL

```bash
cp .env.example .env.local
```

Set:

```bash
TYPEMATTER_SITE_URL=https://docs.example.com
NEXT_PUBLIC_SITE_URL=https://docs.example.com
```

`TYPEMATTER_SITE_URL` is used to generate `robots.txt` and `sitemap.xml`.

## 2. Validate and build

:::steps
1. Run `npm run validate:docs`.
2. Run `npm run build`.
3. Preview with `npm start`.
4. Deploy `out/`.
:::

## Optional Ask AI

The static docs site works without Ask AI. To enable the AI tab, deploy the Worker from `integrations/cloudflare-ask-ai-worker/` and set:

```bash
NEXT_PUBLIC_TYPEMATTER_ASK_AI_ENDPOINT=https://your-worker-domain.workers.dev
NEXT_PUBLIC_TYPEMATTER_ASK_AI_TIMEOUT_MS=25000
```
